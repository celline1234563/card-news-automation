## 2. 6종 그리드 시스템

### 2-1. single-center
```
┌────────────────────────────────┐
│          safe-area 60px        │
│  ┌──────────────────────────┐  │
│  │                          │  │
│  │      content area        │  │
│  │   x:60 y:200 w:960 h:950│  │
│  │                          │  │
│  └──────────────────────────┘  │
│                                │
└────────────────────────────────┘
```
- 용도: hook 임팩트, CTA, 중앙 강조 메시지
- 텍스트 정렬: center
- 헤드라인 최대: 48px (display용은 72px)

### 2-2. top-image-bottom-text
```
┌────────────────────────────────┐
│                                │
│    image / gradient area       │
│    x:0 y:0 w:1080 h:900       │
│    (overlay 필수)              │
│                                │
├────────────────────────────────┤
│  ┌──────────────────────────┐  │
│  │  text area               │  │
│  │  x:60 y:920 w:960 h:370  │  │
│  └──────────────────────────┘  │
└────────────────────────────────┘
```
- 용도: hook 카드, 배경 이미지 활용
- 이미지 위 텍스트 시 반드시 오버레이 적용
- 하단 텍스트 영역 padding: 60px 좌우

### 2-3. left-bar-right-text
```
┌────────────────────────────────┐
│          safe-area 60px        │
│  ┌──┬───────────────────────┐  │
│  │  │                       │  │
│  │8p│   text area           │  │
│  │px│   x:100 y:200         │  │
│  │  │   w:920 h:950         │  │
│  │  │                       │  │
│  └──┴───────────────────────┘  │
└────────────────────────────────┘
```
- 용도: problem, insight, 인용문
- 바 색상: var(--color-primary)
- 텍스트 정렬: left

### 2-4. stat-highlight
```
┌────────────────────────────────┐
│          safe-area 60px        │
│                                │
│  ┌──────────────────────────┐  │
│  │     stat number          │  │
│  │  x:60 y:350 w:960 h:400  │  │
│  │  font-size: 100~120px    │  │
│  │  color: var(--accent)    │  │
│  └──────────────────────────┘  │
│  ┌──────────────────────────┐  │
│  │     label + headline     │  │
│  │  x:60 y:780 w:960 h:400  │  │
│  └──────────────────────────┘  │
└────────────────────────────────┘
```
- 용도: data 카드 전용
- 숫자 font-weight: 900
- 숫자 아래 label: font-size 24px

### 2-5. split-compare
```
┌────────────────────────────────┐
│          safe-area 60px        │
│  ┌───────────┐ ┌───────────┐  │
│  │   left    │ │   right   │  │
│  │  x:60     │ │  x:570    │  │
│  │  w:450    │ │  w:450    │  │
│  │  h:950    │ │  h:950    │  │
│  │           │ │           │  │
│  └───────────┘ └───────────┘  │
└────────────────────────────────┘
```
- 용도: Before/After 비교
- 좌측: 연한 배경, 우측: primary 또는 accent 배경
- 칸 사이 간격: 60px

### 2-6. card-list
```
┌────────────────────────────────┐
│  ┌──────────────────────────┐  │
│  │     header area          │  │
│  │  x:60 y:100 w:960 h:200  │  │
│  └──────────────────────────┘  │
│  ┌──────────────────────────┐  │
│  │  list item 1             │  │
│  │  list item 2             │  │
│  │  list item 3             │  │
│  │  list item 4             │  │
│  │  list item 5             │  │
│  │  x:60 y:320 w:960 h:900  │  │
│  └──────────────────────────┘  │
└────────────────────────────────┘
```
- 용도: solution, summary, insight 스텝
- 항목 높이: 균등 분배 (900 / 항목수)
- 아이콘: 40x40px, 텍스트와 16px 간격

## 3. 배경 처리 유형

| bg_treatment | 설명 | CSS 예시 |
|-------------|------|---------|
| solid_background | 단색 배경 | `background: var(--color-background)` |
| solid_primary | 프라이머리 풀배경 | `background: var(--color-primary)` |
| gradient_subtle | 미세 그라디언트 | `linear-gradient(135deg, var(--color-background), #fff)` |
| gradient_primary | 프라이머리 그라디언트 | `linear-gradient(135deg, var(--color-primary), #1a1a5e)` |
| image_or_gradient_with_overlay | 이미지+오버레이 | `background-image + rgba overlay` |

## 4. 타이포 위계

| 요소 | font-size | font-weight | line-height |
|------|----------|-------------|-------------|
| display (stat 숫자) | 72~120px | 900 | 1.0 |
| h1 (메인 헤드라인) | 48px | 900 | 1.2 |
| h2 (서브 헤드라인) | 36px | 700 | 1.3 |
| h3 (항목 제목) | 28px | 700 | 1.4 |
| body (본문) | 24px | 400 | 1.5 |
| caption (레이블) | 16~20px | 400 | 1.4 |

## 5. 연속 레이아웃 금지 규칙

- 연속 2장에 동일 그리드 사용 금지
- 10장 중 같은 그리드 최대 3회
- single-center는 카드 1, 10에서만 우선 사용
- card-list는 연속 사용 시 아이콘 스타일 변경 필수 (체크 → 숫자 등)
