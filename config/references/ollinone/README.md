# 올인원 수학학원 — 레퍼런스 이미지

이 폴더에 레퍼런스 이미지를 넣어주세요.

## 파일명 규칙

```
{type}-{번호}.png
```

예시:
- `hook-01.png` — 후킹 카드 레퍼런스 1
- `data-01.png` — 데이터 카드 레퍼런스 1
- `cta-01.png` — CTA 카드 레퍼런스 1

## 사용 가능한 type

hook, problem, data, insight, solution, example, summary, cta

## 용도

- Gemini 디자이너에게 시각적 레퍼런스로 전달
- 브랜드 일관성 검토 시 비교 기준
