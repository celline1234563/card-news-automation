# 올인원 수학학원 — 에디토리얼 보이스 가이드

> 이 문서는 researcher.js, hook-critic.js, copywriter.js에서 참조됩니다.

## 1. 말투 원칙

### 1-1. 카테고리별 문체 기준

| 카테고리 | 문체 | 종결어미 | 예시 |
|---------|------|---------|------|
| 입시정보 | 단정체 | ~합니다 / ~입니다 | "2027 수능 수학, 달라지는 3가지입니다" |
| 학습법 | 안내체 | ~하세요 / ~됩니다 | "오답노트는 이렇게 쓰세요" |
| 성과 | 서술체 | ~했습니다 / ~였습니다 | "3개월 만에 4등급에서 2등급이 되었습니다" |
| 학원소식 | 친근체 | ~해요 / ~드려요 | "겨울특강 커리큘럼을 공개해 드려요" |

### 1-2. 톤 스펙트럼

```
[딱딱함] ──────────────── [올인원 위치] ─── [캐주얼]
입시정보 ◀━━━━━━━━━━━ ★ ━━━━━▶ 학원소식
```

- 기본 톤: **전문적이되 차갑지 않은** — 학부모가 "이 학원은 아는 곳이구나" 느끼는 수준
- 절대 금지: 반말, 과도한 이모지, 인터넷 신조어 (ㄹㅇ, 갓, 개꿀 등)

## 2. 헤드라인 공식

### 공식 A — 숫자형
```
[숫자] + [키워드] + [결과/행동]
```
- "중3 수학, <em>3가지</em>만 바꾸면 됩니다"
- "상위 <em>5%</em>가 하는 오답 분석법"
- "수내고 내신, <em>4가지</em> 출제 패턴"

### 공식 B — 질문형
```
[타겟 상황] + [의문] ?
```
- "<em>왜</em> 중2부터 수학이 흔들릴까요?"
- "고1 첫 시험, <em>왜</em> 성적이 떨어질까요?"
- "선행만 하면 <em>정말</em> 괜찮을까요?"

### 공식 C — 충격형
```
[단정적 진술] + [강조 키워드]
```
- "중3 3월, <em>이미 늦었습니다</em>"
- "고등 수학은 <em>중학교와 다릅니다</em>"
- "<em>지금 안 잡으면</em> 고등에서 무너집니다"

## 3. DO / DON'T 예시

### DO (좋은 표현)

| # | 예시 | 이유 |
|---|------|------|
| 1 | "중2 2학기가 고등 수학의 분기점입니다" | 구체적 시점 + 단정적 톤 |
| 2 | "수내고 내신 1등급, 3개월의 기록" | 학교명 + 숫자 + 결과 |
| 3 | "87점에서 시작해 내신 1등급까지" | 구체 수치 + 스토리 |
| 4 | "상위권이 푸는 순서가 다릅니다" | 호기심 유발 + 비교 |
| 5 | "겨울방학 40일, 이렇게 쓰세요" | 숫자 + 행동 유도 |
| 6 | "개념이 아니라 전략이 부족합니다" | 반전 + 인사이트 |
| 7 | "1:4 소수정예, 한 명도 놓치지 않습니다" | 시스템 + 감성 |

### DON'T (피할 표현)

| # | 예시 | 문제점 | 대안 |
|---|------|--------|------|
| 1 | "수학의 신이 되자!" | 과장, 유치함 | "수학 자신감을 되찾는 시간" |
| 2 | "엄마들 주목!!" | 느낌표 과다, 캐주얼 | "학부모님께 알려드립니다" |
| 3 | "수학 잘하는 비밀 대공개~" | 물결표, 가벼운 톤 | "상위권 학생의 공부법을 분석했습니다" |
| 4 | "아이의 미래를 위해" | 진부한 클리셰 | "지금 시작하면 고1 첫 시험이 달라집니다" |
| 5 | "최고의 강사진이 함께합니다" | 근거 없는 최상급 | "평균 경력 12년, 분당 전문 강사진" |
| 6 | "성적 폭발적 향상!!" | 과장 + 느낌표 | "평균 27점 상승, 3개월 집중 결과" |
| 7 | "우리 아이 수학 걱정 끝!" | 근거 없는 단정 | "걱정에서 자신감으로, 올인원의 3단계 시스템" |

## 4. 금지 표현 리스트

### 4-1. 진부한 클리셰
- "아이의 미래를 위해"
- "꿈을 향한 첫 걸음"
- "성공의 지름길"
- "수학의 왕도"
- "학부모님의 현명한 선택"
- "우리 아이 수학 천재 만들기"
- "수포자 탈출"

### 4-2. 과장·공격 표현
- "폭발적", "미친", "역대급", "레전드"
- "최고의", "최강의", "압도적" (근거 없는 최상급)
- "지금 안 하면 끝장납니다" (공포 마케팅 과잉)
- "다른 학원은 이런 거 못 합니다" (경쟁사 비하)
- "100% 성적 향상 보장" (허위 약속)

### 4-3. 캐주얼 표현
- 물결표 (~)를 문장 끝에 사용
- 느낌표 2개 이상 (!!, !!!)
- "ㅋㅋ", "ㅎㅎ", "ㄹㅇ" 등 인터넷 용어
- "고고", "레츠고", "화이팅" 등 경쾌한 구호

## 6. 문장 작성 원칙

### 6-1. 간결성
- 헤드라인: **30자 이내** (16자 이내가 이상적)
- 서브텍스트: **80자 이내**
- 한 문장에 하나의 메시지만

### 6-2. 구체성 우선
```
나쁜 예: "수학 성적을 올리세요"
좋은 예: "중간고사 수학 27점 올리는 3단계"
```

### 6-3. 리듬감
- 3단 구성을 활용: "진단 → 설계 → 실행"
- 대구법 활용: "중학교에서 잘했다고, 고등학교도 잘한다는 보장은 없습니다"
- 짧은 문장 + 긴 문장 교차

### 6-4. 강조(<em>) 사용법
- 한 헤드라인에 <em> 태그는 **1~2개**만
- 핵심 키워드만 강조 (부사/조사 강조 금지)
- 숫자, 학교명, 핵심 동사에 사용

## 7. researcher.js 연동 설계

### 7-1. emotion-curve.json 주입 방식

researcher 프롬프트에 아래 형태로 감정곡선을 주입합니다:

```
## 감정곡선 가이드

각 카드는 아래 감정 목표를 따르세요:

카드 1 (hook): 감정=충격, 강도=9/10
카드 2 (problem): 감정=공감, 강도=7/10
...
카드 10 (cta): 감정=행동촉구, 강도=8/10
```

### 7-2. 카드 JSON에 emotion_target 필드 추가

```json
{
  "number": 1,
  "type": "hook",
  "emotion_target": "충격",
  "emotion_intensity": 9,
  ...
}
```

- researcher.js가 생성 시 자동으로 emotion-curve.json에서 매핑
- hook-critic.js에서 emotion_target을 채점 참고 정보로 활용
- gemini-designer.js에서 감정에 맞는 비주얼 톤 결정에 참고
