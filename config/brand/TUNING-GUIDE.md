# 브랜드 파인튜닝 가이드
> 이 가이드는 카드뉴스 결과물의 톤·디자인·카피를 조절하고 싶을 때 참고하는 문서입니다.
> **코드를 수정할 필요 없이**, 아래 파일들만 편집하면 결과물이 바뀝니다.

---

## 어떤 파일을 수정하면 되나요?

| 바꾸고 싶은 것 | 수정할 파일 | 예시 |
|---------------|-----------|------|
| 카피 말투·헤드라인 스타일 | `{학원}-voice.md` | "좀 더 부드럽게 말했으면" → voice.md의 톤 스펙트럼 조절 |
| 타겟 고객·콘텐츠 방향 | `{학원}-strategy.md` | "고등부도 추가됐어" → strategy.md의 페르소나 추가 |
| 디자인 색상·그림자·여백 | `{학원}-tokens.json` | "글씨가 너무 커" → tokens.json의 font size 조절 |
| 카드별 감정 흐름 | `emotion-curve.json` | "첫 카드가 너무 세" → intensity 값 낮추기 |
| 배경 이미지 스타일 | `imager-system.txt` | "배경이 너무 어두워" → 무드 키워드 변경 |

---

## 파일별 수정 가이드

### 1. `{학원}-strategy.md` — 브랜드 전략

**이걸 바꾸면**: AI가 기획하는 주제·톤·타겟이 바뀝니다.

#### 자주 수정하는 부분

| 섹션 | 언제 수정 | 어떻게 |
|------|----------|--------|
| **1-3. 페르소나** | 타겟 학부모가 바뀌었을 때 | 이름·나이·고민·반응 메시지를 실제 상담 사례 기반으로 수정 |
| **2-1. 무드 키워드** | 디자인 분위기를 바꾸고 싶을 때 | 키워드 3개를 교체 (예: "진중한" → "모던한") |
| **2-2. 금지 키워드** | 불필요한 제한이 있을 때 | 항목 추가/삭제 |
| **3-1. 콘텐츠 카테고리** | 주력 콘텐츠가 바뀌었을 때 | 비중(%) 조절, 카테고리 추가 |
| **4. 타겟 학교** | 타겟 학교가 추가/변경될 때 | 학교명 추가/삭제 |

#### 수정 예시

```markdown
# 변경 전
| 반응하는 메시지 | "중2 2학기가 고등 수학의 분기점입니다" |

# 변경 후 (실제 상담에서 먹히는 문구로 교체)
| 반응하는 메시지 | "중2 겨울방학이 고등 수학의 마지막 기회입니다" |
```

### 2. `{학원}-voice.md` — 말투·표현 가이드

**이걸 바꾸면**: AI가 쓰는 헤드라인·카피의 말투가 바뀝니다.

#### 자주 수정하는 부분

| 섹션 | 언제 수정 | 어떻게 |
|------|----------|--------|
| **2. 헤드라인 공식** | "이런 스타일의 헤드라인이 좋다" | 공식과 예시를 추가/교체 |
| **3. DO/DON'T** | 결과물에 안 맞는 표현이 나올 때 | DON'T에 추가하고 대안 작성 |
| **4. 금지 표현** | 특정 표현이 계속 나올 때 | 금지 리스트에 추가 |

#### 수정 예시

```markdown
# DON'T에 추가
| 8 | "수학이 재밌어집니다" | 근거 없는 감정 | "수학 문제 풀이 시간이 절반으로 줄어듭니다" |
```

**팁**: 실제 카드뉴스 결과물에서 "이건 별로다" 싶은 표현이 보이면, 그 표현을 DON'T에 바로 추가하세요.

### 3. `{학원}-tokens.json` — 디자인 토큰

**이걸 바꾸면**: 카드뉴스의 글씨 크기, 색상, 그림자, 여백이 바뀝니다.

#### 자주 수정하는 부분

| 항목 | 위치 | 설명 |
|------|------|------|
| 글씨 크기 | `typography.size` | h1=48, body=24 등. 숫자(px)를 조절 |
| 색상 | `color.primary` 등 | HEX 코드 변경. **academies.json의 theme도 같이 바꿔야 함** |
| 오버레이 투명도 | `effects.overlay` | rgba의 마지막 숫자(0~1)로 투명도 조절. 0.7=70% 불투명 |
| 그림자 | `effects.shadow` | subtle/medium/strong 단계별 그림자 강도 |
| 안전 영역 | `spacing.canvas.safe_area` | 카드 가장자리 여백 (기본 60px) |

#### 수정 예시

```json
// 헤드라인이 너무 크다면
"h1": 48  →  "h1": 42

// 오버레이가 너무 어둡다면
"overlay_dark": "rgba(26, 26, 46, 0.7)"  →  "rgba(26, 26, 46, 0.5)"
```

**주의**: `color.primary` 등 메인 컬러를 바꿀 때는 반드시 `config/academies.json`의 `theme` 섹션도 같은 값으로 수정하세요.

### 4. `emotion-curve.json` — 감정 곡선

**이걸 바꾸면**: 카드 10장의 감정 흐름이 바뀝니다.

| 필드 | 설명 | 조절 방법 |
|------|------|----------|
| `emotion` | 해당 카드의 감정 목표 | "충격" → "호기심" 등으로 교체 |
| `intensity` | 감정 강도 (1~10) | 숫자를 높이면 더 강하게, 낮추면 부드럽게 |
| `description` | AI에게 주는 설명 | 더 구체적으로 쓸수록 결과가 좋음 |

#### 수정 예시

```json
// 1번 카드가 너무 자극적이라면
{ "card": 1, "emotion": "충격", "intensity": 9 }
→
{ "card": 1, "emotion": "호기심", "intensity": 7 }
```

### 5. `imager-system.txt` — 이미지 스타일 전략

**이걸 바꾸면**: Imagen이 생성하는 배경 이미지의 무드·스타일이 바뀝니다.

#### 자주 수정하는 부분

| 항목 | 언제 수정 | 어떻게 |
|------|----------|--------|
| **카드 타입별 무드** | 특정 카드의 배경 분위기를 바꾸고 싶을 때 | 테이블에서 해당 type의 무드 설명 수정 |
| **공통 스타일** | 전체적인 사진 느낌을 바꾸고 싶을 때 | "기본 스타일:" 뒤의 키워드 교체 |
| **금지 사항** | 이미지에 특정 요소를 추가/제외하고 싶을 때 | "금지:" 줄에 항목 추가 |

#### 수정 예시

```
# hook 카드 배경을 매거진 느낌으로 바꾸고 싶다면
| hook | 임팩트 있는, 시선을 사로잡는 |
→
| hook | 매거진풍, 세련되고 모던한 |

# 전체 스타일을 바꾸고 싶다면
기본 스타일: 미니멀, 따뜻한 조명, 부드러운 색감
→
기본 스타일: 매거진풍, 자연광, 선명한 색감
```

**위치**: `prompts/imager-system.txt`

## 수정 후 반영 방법

1. 파일 수정 후 저장
2. Git에 커밋 & 푸시
   ```
   git add config/brand/ config/tokens/
   git commit -m "브랜드 설정 업데이트: [변경 내용]"
   git push
   ```
3. Railway가 자동 배포 → 다음 카드뉴스부터 반영됨

**즉시 테스트하고 싶다면**:
```
node index.js "테스트 주제" --academy jinhak
```
로컬에서 결과물을 바로 확인할 수 있습니다.

## FAQ

**Q: 결과물에서 특정 표현이 계속 나와요**
→ `voice.md`의 "금지 표현 리스트"에 해당 표현을 추가하세요.

**Q: 디자인 색상을 바꾸고 싶어요**
→ `tokens.json`의 color 섹션 + `academies.json`의 theme 섹션을 같이 수정하세요.

**Q: 새 학원을 추가하고 싶어요**
→ `academies.json`에 학원 추가 + `{학원}-strategy.md`, `{학원}-voice.md`, `{학원}-tokens.json` 3개 파일을 새로 만드세요. 기존 학원 파일을 복사해서 내용만 바꾸면 됩니다.

**Q: emotion-curve.json을 학원별로 다르게 하고 싶어요**
→ 현재는 공용입니다. 학원별로 분리하려면 개발자에게 요청하세요.
