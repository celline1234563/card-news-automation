## 1. 학원 정체성

### 1-1. 핵심 가치

**"스마트 영어 — 국제학교·특목고를 향한 똑부러지는 영어 설계"**

톡톡 잉글리쉬는 단순 회화나 시험 영어가 아니라, 국제학교·특목고·화상영어·미국교과서를 아우르는 **전문 초등 영어 학원**이다.

| 가치 | 한 문장 정의 |
|------|-------------|
| 전문 영어 | 국제학교·특목고 대비 — 일반 보습과 다른 레벨 |
| 실전 활용 | 미국교과서·화상영어 — 진짜 쓰는 영어를 가르친다 |
| 깔끔한 관리 | 커리큘럼·테스트·피드백이 체계적으로 정리됨 |
| 똑부러지는 결과 | 수치로 증명 — 시험 점수, 레벨 테스트, 실력 변화 |

### 1-2. 경쟁 포지셔닝

초등 영어 시장은 대형 프랜차이즈(윤선생, 청담어학원)와 화상영어 플랫폼이 양분하고 있다.

| 경쟁 유형 | 특징 | 톡톡 차별점 |
|-----------|------|-----------|
| 대형 영어 프랜차이즈 | 표준화된 교재, 그룹 수업 | **미국교과서 기반** — 국제학교 수준의 커리큘럼 |
| 화상영어 플랫폼 | 저비용, 원어민이지만 관리 부재 | **화상영어 + 오프라인 관리** — 온·오프 하이브리드 |
| 동네 영어학원 | 밀착되지만 특목고 대비 약함 | **특목고 입시 전문성** — 면접·에세이·포트폴리오 준비 |
| 영어유치원 출신 | 유아기 영어 기반 있지만 유지 어려움 | **연속성 유지** — 영유 이후 레벨에 맞는 심화 과정 |

**포지셔닝 한 줄**: 국제학교 수준의 전문성 + 동네 학원의 밀착관리 = **"스마트 초등 영어 전문"**

## 2. 비주얼 원칙

### 2-1. 무드 키워드 (확정)

| 순서 | 키워드 | 의미 | 디자인 반영 |
|------|--------|------|-----------|
| 1 | **깔끔한** | 정돈된, 여백 활용, 군더더기 없는 | 미니멀 레이아웃, 충분한 패딩, 적은 요소 |
| 2 | **스마트한** | 전문적, 체계적, 데이터 기반 | 차트형 레이아웃, 숫자 강조, 명확한 위계 |
| 3 | **똑부러지는** | 명확한, 자신감 있는, 결과 중심 | 강한 CTA, 깔끔한 타이포, 명료한 구조 |

### 2-2. 금지 키워드 — 절대 쓰지 않을 톤

| 금지 키워드 | 이유 | 디자인에서 피할 것 |
|------------|------|------------------|
| 유치한 | 초등이지만 학부모 타겟 — 교육 전문성 필요 | 만화 캐릭터, 유아용 폰트, 말풍선 남발 |
| 무거운 | 초등 영어의 경쾌함과 충돌 | 어두운 톤 위주, 딱딱한 레이아웃 |
| 복잡한 | 한 눈에 안 들어오는 디자인 | 요소 과밀, 정보 과부하, 3+색 포인트 |
| 촌스러운 | 국제학교·특목고 지향 학부모 기대에 미달 | 클립아트, 저해상도, 촌스러운 그라데이션 |
| 과시적 | 가벼운 자랑은 반감 유발 | "최고" "최강" 남발, 근거 없는 화려함 |

### 2-3. 레퍼런스 방향

**모던 에듀테크 + 깔끔한 매거진**

| 레퍼런스 스타일 | 참고 포인트 |
|---------------|-----------|
| 에듀테크 브랜드 | 깔끔한 UI, 밝은 톤, 진행률·레벨 시각화 |
| 글로벌 교육 서비스 | 영어 환경 느낌, 모던하고 세련된 레이아웃 |
| 미니멀 매거진 | 여백 > 요소, 한 장에 하나의 메시지 |
| 라이프스타일 브랜드 | 밝고 따뜻한 컬러, 활기찬 느낌 |

**피해야 할 레퍼런스**: 유아 교육 (파스텔+캐릭터), 이벤트 전단지, 어두운 입시 분위기

### 2-4. 컬러 의도 + 사용 비율

#### 컬러 팔레트

| 역할 | 컬러 | HEX | 의도 |
|------|------|-----|------|
| Primary | 비비드 오렌지 | `#FF6B2B` | 활력, 에너지, 자신감 — 초등 영어의 밝은 이미지 |
| Secondary | 화이트 | `#FFFFFF` | 깔끔함, 여백, 가독성 극대화 |
| Background | 웜 화이트 | `#FFF8F5` | 따뜻한 배경, 부드러운 느낌 |
| Text | 블랙 | `#1A1A1A` | 본문 가독성, 명료함 |
| Highlight | 소프트 피치 | `#FFD4B8` | 텍스트 하이라이트, 배지 배경 |
| Accent | 딥 오렌지 | `#E55A1B` | 버튼, 아이콘, CTA 강조 |

#### 사용 비율 가이드

```
전체 화면 기준:
  Neutral (화이트~웜) ████████████████░░░░░░░░░░░░░░  55%  — 배경, 여백, 깔끔한 공간
  Primary (#FF6B2B)  ████████████░░░░░░░░░░░░░░░░░░  35%  — 헤더, 강조, 키 비주얼
  Highlight (#FFD4B8) ███░░░░░░░░░░░░░░░░░░░░░░░░░░░  10%  — 포인트, 하이라이트
```

#### 컬러 적용 규칙

| 상황 | 사용 컬러 | 금지 |
|------|----------|------|
| 강조 텍스트 | Primary만 | Highlight로 텍스트 강조 금지 |
| 배경 포인트 영역 | Highlight만 | Primary를 넓은 배경에 과다 사용 시 눈 피로 |
| 숫자/통계 | Accent (=딥 오렌지) | 빨강·초록 등 외부 색상 금지 |
| 다크 배경 카드 | Primary 배경 + 흰색 텍스트 | Highlight 배경에 다크 텍스트 금지 |
| Secondary + Accent | — | **동시 사용 절대 금지** (CLAUDE.md 규칙) |

## 3. 콘텐츠 방향

### 3-1. 주력 콘텐츠 카테고리

| 카테고리 | 비중 | 설명 | 예시 주제 |
|---------|------|------|----------|
| **입시·진로 정보** | 25% | 국제학교, 특목고, 영어 교육 트렌드 | "국제학교 편입 준비 체크리스트", "외고 면접 영어, 어떻게 준비할까" |
| **학습법** | 30% | 영어 공부법, 리딩·라이팅·스피킹 팁 | "미국교과서로 배우는 리딩 전략", "화상영어 효과 200% 올리는 법" |
| **성과** | 25% | 레벨 향상, 시험 결과, 학생 후기 | "6개월 만에 리딩 레벨 2단계 UP", "톡톡 수강생 영어 시험 결과" |
| **학원소식** | 20% | 수업 소개, 커리큘럼, 이벤트 | "톡톡 화상영어 시스템 소개", "겨울 집중 캠프 안내" |

### 3-2. 카테고리별 톤 차이

| 카테고리 | 감정 톤 | 말투 특징 | 헤드라인 스타일 |
|---------|---------|----------|---------------|
| 입시·진로 | **전략적** — "이렇게 준비하면 된다" | 구체적, 로드맵 제시, 자신감 있는 | 방향형: "국제학교 편입, 초3부터 시작하세요" |
| 학습법 | **실용적** — "이 방법이 효과적이다" | 스텝 제시, 팁 공유, 명확한 결과 | 방법형: "미국교과서 리딩, 3단계로 정복합니다" |
| 성과 | **깔끔한 자신감** — "이렇게 됐다" | 수치 제시, 절제된 어조, 사실 기반 | 결과형: "리딩 레벨 2단계 UP, 6개월의 변화" |
| 학원소식 | **활기찬** — "이런 곳이에요" | 시스템 설명, 밝은 마무리 | 소개형: "톡톡이 영어를 가르치는 방법" |

### 3-3. 게시 빈도·요일 가이드라인

| 항목 | 기준 |
|------|------|
| 주간 게시 수 | 2~3회 |
| 권장 요일 | 화·목·토 |
| 화요일 | 학습법 or 입시정보 (주중 정보 탐색) |
| 목요일 | 성과 or 학습법 (주 후반 동기부여) |
| 토요일 | 학원소식 or 성과 (주말 여유 시간) |
| 게시 시간 | 오전 8~9시 or 오후 9~10시 (학부모 인스타 활성) |
