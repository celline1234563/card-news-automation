## 1. 말투 원칙

### 1-1. 카테고리별 문체 기준

| 카테고리 | 문체 | 종결어미 | 예시 |
|---------|------|---------|------|
| 입시정보 | 단정체 | ~합니다 / ~입니다 | "중2 수학, 지금 놓치면 고등이 흔들립니다" |
| 학습법 | 안내체 | ~하세요 / ~됩니다 | "매일 15분, 이렇게 복습하세요" |
| 성과 | 서술체 | ~했습니다 / ~였습니다 | "60점에서 시작해 85점이 되었습니다" |
| 학원소식 | 친근체 | ~해요 / ~드려요 | "진학학원의 관리 시스템을 소개해 드려요" |

### 1-2. 톤 스펙트럼

```
[딱딱함] ─────────── [진학 위치] ──────── [캐주얼]
입시정보 ◀━━━━━━━━ ★ ━━━━━━━━▶ 학원소식
```

- 기본 톤: **대담하면서 따뜻한** — "이 학원은 확실히 해주겠구나" 느끼는 수준
- 올인원 대비: 약간 더 따뜻하고, 초등 학부모를 위한 부드러움 포함
- 절대 금지: 반말, 과도한 이모지, 인터넷 신조어 (ㄹㅇ, 갓, 개꿀 등)

## 2. 헤드라인 공식

### 공식 A — 숫자형
```
[숫자] + [키워드] + [결과/행동]
```
- "초5 수학, <em>3가지</em>만 잡으면 중학교가 쉽습니다"
- "시흥중 내신 <em>4가지</em> 출제 패턴"
- "매일 <em>15분</em>, 수학 습관이 바뀝니다"

### 공식 B — 질문형
```
[타겟 상황] + [의문] ?
```
- "<em>왜</em> 초등학교 때 잘하던 수학이 중학교에서 무너질까요?"
- "학원을 옮겨도 <em>성적이 안 오르는</em> 이유는?"
- "선행만 하면 <em>정말</em> 실력이 될까요?"

### 공식 C — 대담형
```
[강한 진술] + [해결 키워드]
```
- "수학을 포기하려는 아이, <em>3개월이면 바뀝니다</em>"
- "성적이 안 오르는 건 <em>방법이 틀렸기</em> 때문입니다"
- "<em>빈 곳만 채우면</em> 수학은 다시 올라갑니다"

## 3. DO / DON'T 예시

### DO (좋은 표현)

| # | 예시 | 이유 |
|---|------|------|
| 1 | "초5부터 잡으면 중학교 수학이 쉬워집니다" | 구체적 시점 + 결과 |
| 2 | "시흥중 내신 분석, 출제 패턴 4가지" | 학교명 + 숫자 + 구체성 |
| 3 | "60점에서 85점, 3개월의 기록" | 구체 수치 + 스토리 |
| 4 | "학원을 옮겨도 안 되는 이유" | 호기심 유발 + 문제 공감 |
| 5 | "매일 15분, 수학 습관이 달라집니다" | 숫자 + 실천 가능한 행동 |
| 6 | "개념이 아니라 빈 곳이 문제입니다" | 반전 + 인사이트 |
| 7 | "한 명도 놓치지 않는 관리 시스템" | 맞춤 + 신뢰감 |

### DON'T (피할 표현)

| # | 예시 | 문제점 | 대안 |
|---|------|--------|------|
| 1 | "수학 천재 만들기 프로젝트!" | 과장, 유치함 | "수학 자신감을 되찾는 3단계" |
| 2 | "엄마 아빠 주목~!!" | 느낌표 과다, 캐주얼 | "학부모님께 알려드립니다" |
| 3 | "성적 폭발적 상승!!!" | 과장, 근거 없음 | "평균 25점 상승, 3개월 집중 결과" |
| 4 | "우리 아이 수학 걱정 끝!" | 근거 없는 단정 | "걱정을 자신감으로 바꾸는 진학 시스템" |
| 5 | "최고의 강사진" | 근거 없는 최상급 | "평균 경력 10년, 관악 전문 강사진" |
| 6 | "아이의 밝은 미래를 위해" | 진부한 클리셰 | "지금 시작하면 중간고사가 달라집니다" |

## 4. 금지 표현 리스트

### 4-1. 진부한 클리셰
- "아이의 미래를 위해"
- "꿈을 향한 첫 걸음"
- "수학의 왕도"
- "학부모님의 현명한 선택"
- "수포자 탈출 프로젝트"

### 4-2. 과장·공격 표현
- "폭발적", "미친", "역대급", "레전드"
- "최고의", "최강의", "압도적" (근거 없는 최상급)
- "지금 안 하면 끝장납니다" (공포 마케팅 과잉)
- "다른 학원은 이런 거 못 합니다" (경쟁사 비하)
- "100% 성적 향상 보장" (허위 약속)

### 4-3. 캐주얼 표현
- 물결표 (~)를 문장 끝에 사용
- 느낌표 2개 이상 (!!, !!!)
- "ㅋㅋ", "ㅎㅎ", "ㄹㅇ" 등 인터넷 용어
- "고고", "레츠고", "화이팅" 등 경쾌한 구호

## 6. 문장 작성 원칙

### 6-1. 간결성
- 헤드라인: **30자 이내** (16자 이내가 이상적)
- 서브텍스트: **80자 이내**
- 한 문장에 하나의 메시지만

### 6-2. 구체성 우선
```
나쁜 예: "수학 성적을 올리세요"
좋은 예: "중간고사 수학 25점 올리는 3단계"
```

### 6-3. 리듬감
- 3단 구성을 활용: "진단 → 맞춤 → 관리"
- 대구법 활용: "초등학교에서 잘했다고, 중학교도 잘한다는 보장은 없습니다"
- 짧은 문장 + 긴 문장 교차

### 6-4. 강조(<em>) 사용법
- 한 헤드라인에 <em> 태그는 **1~2개**만
- 핵심 키워드만 강조 (부사/조사 강조 금지)
- 숫자, 학교명, 핵심 동사에 사용
