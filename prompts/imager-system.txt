# Imagen 배경 이미지 프롬프트 전략
> 이 파일은 카드뉴스 배경 이미지 생성 시 Imagen에 전달할 프롬프트를 구성합니다.
> 직원이 이미지 스타일을 튜닝하려면 이 파일만 수정하면 됩니다.

---

## 공통 스타일 (모든 카드에 적용)

기본 스타일: 미니멀, 따뜻한 조명, 부드러운 색감
구도: 세로형 (3:4 비율)
필수: 텍스트 가독성을 위해 중앙 영역은 밝고 흐릿하게 처리
금지: 텍스트, 글자, 로고, 워터마크 절대 포함 금지

---

## 카드 타입별 무드

| type | 무드 | 설명 |
|------|------|------|
| hook | 임팩트 있는, 시선을 사로잡는 | 강렬한 첫인상, 스크롤 멈추게 |
| problem | 공감을 자아내는, 학부모의 고민을 담은 | 걱정, 고민, 현실적 분위기 |
| empathy | 공감을 자아내는, 학부모의 고민을 담은 | 걱정, 고민, 현실적 분위기 |
| data | 깔끔하고 전문적인, 데이터 시각화에 적합한 | 신뢰감, 객관적 느낌 |
| insight | 깨달음의 순간, 밝아지는 느낌 | 이해의 전환점 |
| solution | 밝고 희망적인, 해결의 느낌 | 긍정적, 행동 유도 |
| example | 성공적인, 증명하는 | 실제 사례, 결과물 느낌 |
| summary | 정돈된, 깔끔한 | 핵심 정리, 미니멀 |
| cta | 따뜻하고 신뢰감 있는, 행동을 유도하는 | 마무리, 초대하는 분위기 |

---

## 프롬프트 템플릿

```
한국 학원 인스타그램 카드뉴스 배경 이미지.
주제: {{HEADLINE}}
카테고리: {{IMAGE_CATEGORY}}
무드: {{MOOD}}
스타일: {{STYLE}}
중요: {{CONSTRAINT}}
색감: {{COLOR_TONE}} 계열 톤
세로형 구도 (3:4 비율)
```

## 수정 가이드

### 무드를 바꾸고 싶을 때
위 "카드 타입별 무드" 테이블에서 해당 type의 무드 설명을 수정하세요.
예: hook의 "임팩트 있는" → "세련되고 모던한" 으로 변경

### 스타일을 바꾸고 싶을 때
"공통 스타일"의 "기본 스타일" 줄을 수정하세요.
예: "미니멀, 따뜻한 조명, 부드러운 색감" → "매거진풍, 자연광, 선명한 색감"

### 특정 카테고리에 추가 지시를 넣고 싶을 때
"카드 타입별 무드" 테이블에 설명을 더 구체적으로 작성하세요.
예: solution의 설명 → "긍정적, 행동 유도, 밝은 자연광이 들어오는 공부방"
