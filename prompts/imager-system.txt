# Imagen 배경 이미지 프롬프트 전략
> 이 파일은 카드뉴스 배경 이미지 생성 시 Imagen에 전달할 프롬프트를 구성합니다.
> 직원이 이미지 스타일을 튜닝하려면 이 파일만 수정하면 됩니다.

---

## 핵심 원칙: 이미지는 독립 영역

이미지는 텍스트와 **물리적으로 분리된 영역**에 배치됩니다.
따라서 "텍스트 가독성을 위한 흐림 처리" 같은 건 필요 없습니다.
이미지 자체가 아름답고 주제에 맞으면 됩니다.

---

## 공통 스타일 (모든 카드에 적용)

기본 스타일: 미니멀, 고품질, 매거진급 사진
구도: 세로형 (3:4 비율), 피사체가 중앙에 명확하게
필수: 피사체 하나에 집중. 깨끗한 배경. 고해상도 느낌
금지: 텍스트, 글자, 로고, 워터마크, 사람 얼굴, 복잡한 배경

---

## 카드 타입별 무드

| type | 무드 | 피사체 예시 |
|------|------|-----------|
| hook | 시선을 사로잡는, 강렬한 | 극적인 조명의 책상/펜/노트, 결단의 순간 |
| problem | 차갑고 무거운 분위기 | 어두운 톤의 교실, 빈 책상, 흐린 날씨 |
| empathy | 따뜻하지만 걱정스러운 | 저녁 조명의 책상, 고민하는 분위기의 정물 |
| data | 깔끔하고 전문적인 | 그래프 옆 노트, 깔끔한 사무용품, 미니멀 오브젝트 |
| insight | 밝아지는, 깨달음 | 창문으로 들어오는 빛, 전구, 밝은 톤 |
| solution | 밝고 희망적인 | 정돈된 책상, 체크리스트, 깔끔한 학습 도구 |
| example | 성공적인, 증거 | 성적표, 트로피, 메달, 합격 느낌 |
| summary | 정돈된, 미니멀 | 깔끔하게 정리된 노트, 미니멀 오브젝트 |
| cta | 따뜻하고 초대하는 | 열린 문, 밝은 복도, 환영하는 공간 |

---

## 색감 규칙

- 학원 브랜드 컬러 계열 톤으로 통일
- 채도 낮추고 톤 맞추기 (과도한 원색 금지)
- 10장이 나란히 놓였을 때 색감이 통일돼야 함
- 같은 시리즈 내에서 너무 다른 색감의 이미지가 나오면 안 됨

---

## 프롬프트 템플릿

```
한국 학원 인스타그램 카드뉴스용 사진.
피사체: {{IMAGE_CATEGORY}}
무드: {{MOOD}}
스타일: 미니멀, 고품질, 매거진급. 피사체 하나에 집중. 깨끗한 배경.
색감: {{COLOR_TONE}} 계열 톤, 채도 낮은 차분한 색감
중요: 텍스트, 글자, 로고, 워터마크, 사람 얼굴 절대 포함 금지. 피사체가 중앙에 명확하게.
세로형 구도 (3:4 비율)
```

## 수정 가이드

### 무드를 바꾸고 싶을 때
위 "카드 타입별 무드" 테이블에서 해당 type의 무드와 피사체 예시를 수정하세요.

### 스타일을 바꾸고 싶을 때
"공통 스타일"의 "기본 스타일" 줄을 수정하세요.
예: "미니멀, 고품질, 매거진급 사진" → "수채화풍, 부드러운 파스텔 톤"

### 색감을 바꾸고 싶을 때
"색감 규칙" 섹션을 수정하세요. 채도, 톤, 온도 등을 구체적으로 명시하면 됩니다.
