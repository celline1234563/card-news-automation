# 카드뉴스 자동화 시스템 — 운영 가이드

## 전체 흐름

```
노션 콘텐츠 캘린더에서 상태 변경
  → 시스템 자동 감지 (30초 간격)
  → AI 기획/디자인 생성
  → Google Drive 업로드 (PNG + HTML)
  → 노션에 이미지 삽입
  → 디자이너가 Figma 플러그인으로 Import → 편집
```

## Figma 작업 방법

### 사전 준비 (최초 1회)

**1. 카드뉴스 Import 플러그인 설치**

1. Figma 데스크탑 앱 실행
2. 좌측 메뉴 → **Plugins** → **Development** → **Import plugin from manifest...**
3. 파일 선택: `cardnews/figma-plugin/manifest.json`
4. "카드뉴스 Import" 플러그인이 Development 목록에 추가됨

### 카드 Import (매 작업)

**1. 로컬 서버 실행** (터미널)

```bash
node server.js
```
> `📡 카드뉴스 API 서버: http://localhost:3456` 메시지 확인

**2. Figma에서 플러그인 실행**

1. Figma 파일 열기
2. `Cmd+/` (Mac) → "카드뉴스 Import" 검색 → Run
3. 또는: 우클릭 → Plugins → Development → 카드뉴스 Import

**3. 카드 Import**

1. 서버 주소 `http://localhost:3456` 확인 → **연결** 클릭
2. 배치 목록에서 원하는 카드뉴스 선택
3. 카드 10장 미리보기 확인
4. **🚀 Figma에 10장 Import** 클릭
5. 프로그레스 바 완료 → Figma 캔버스에 10개 프레임 자동 생성

**4. 편집**

- 각 프레임(1080×1350px)이 카드 이미지로 채워져 있음
- 새 페이지("카드뉴스")에 카드 10장이 나란히 배치됨
- 텍스트, 색상, 레이아웃 자유롭게 수정
- 인스타그램 세로형 1080×1350px 크기 유지 권장

### 편집 가능한 레이어로 변환 (선택)

PNG 이미지를 편집 가능한 벡터 레이어로 바꾸려면:

1. Figma에서 **html.to.design** 플러그인 설치 (Community → 검색)
2. 노션 페이지 → "📋 HTML 소스 파일" 토글에서 card-XX.html 열기
3. HTML 코드 복사 → html.to.design Editor 탭 → 붙여넣기 → Create
4. 텍스트/색상/레이아웃이 개별 Figma 레이어로 변환됨

## 노션 콘텐츠 캘린더 사용법

### 새 카드뉴스 요청

1. 노션 콘텐츠 캘린더에서 새 페이지 생성
2. 제목: `[학원접두사 과목] 주제` (예: `[올인원 중등3-2] 중간고사 수학 준비법`)
3. 상태를 **기획 착수**로 설정
4. (선택) 메인 키워드, 작업기획서 섹션에 참고사항 작성
5. 시스템이 자동으로 기획안 생성 → 상태가 **기획 컨펌**으로 변경됨

### 디자인 요청

1. 기획안 확인 후 상태를 **제작 요청**으로 변경
2. 시스템이 PNG 10장 + 원고 생성 → Drive 업로드
3. 완료되면 상태가 **디자인 1차**로 변경됨

### 수정 요청

1. 댓글에 `@수정 [수정 지시사항]` 작성
   - 예: `@수정 카드 3번 통계 수치를 최신 데이터로 변경해주세요`
2. 상태를 **디자인 수정**으로 변경
3. 시스템이 수정 지시 반영 후 재생성 → 상태가 **디자인 수정 완료**로 변경됨

## 문제 해결

| 증상 | 원인 | 해결 |
|------|------|------|
| 플러그인 "연결 실패" | 서버 미실행 | `node server.js` 실행 |
| 노션 이미지 안 뜸 | Drive 권한 문제 | Drive 공유 드라이브에 서비스 계정 멤버 확인 |
| 기획안이 안 생김 | API 크레딧 부족 | console.anthropic.com에서 충전 |
| 학원이 인식 안 됨 | 접두사 불일치 | academies.json의 notion_prefix 확인 |
| Figma 플러그인 안 보임 | 설치 안 됨 | Plugins → Development → Import manifest |
