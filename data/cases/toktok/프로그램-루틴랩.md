## 왜 좋은 영어 습관이 필요할까?

> (특목고 · 자사고 · 서울대 사례로 알아보기)

- 특목고/자사고 입시에서 영어는 **핵심 경쟁력**
- 외고·국제고는 내신 평가에 **영어 점수가 가장 중요한 요소**로 반영
- 교과 내신 A등급 비중이 특목고/자사고 **40% 이상** (일반고의 **2배**)
- 서울대·명문대 합격생들의 공통점 = **어릴 때부터 꾸준한 영어 노출과 자기주도 습관**

> **"지금 시작하는 루틴이 우리 아이의 학업 미래와 연결됩니다"**

## 루틴랩 철학

> **"공부의 주인은 '선생님'이 아닌, '우리아이'입니다."**

- 주도는 우리아이가 **스스로 루틴을 짜나가게** 도와주는 프로그램
- 아이가 변화하는데 걸리는 시간은 **단 4주**
- "세살버릇 여든까지 간다"는 속담처럼, 지금 잡아놓은 좋은 루틴이 **평생 학습을 결정**

### 루틴랩 4가지 키워드
- 자기주도 학습 습관 형성
- 비용 효율적인 영어 학습
- 체계적인 5가지 챌린지
- 개인 맞춤형 루틴 제공

## 루틴랩 사용설명서 — 5단계 프로세스

```
① 등원 → ② 학습 시작 → ③ 1:1 피드백 → ④ 귀가 → ⑤ 반복
```

| 단계 | 내용 |
|------|------|
| **1. 등원** | 원하는 시간대에 루틴랩에 등원합니다 |
| **2. 학습 시작** | 선택한 챌린지 자체교재로 학습을 시작합니다 |
| **3. 1:1 피드백** | 틀린 부분을 1:1로 피드백 받고 교정합니다 |
| **4. 귀가** | 학습 완료 후 귀가하며 성취감을 느낍니다 |
| **5. 반복** | 월 12회 완주까지 반복하여 습관을 형성합니다 |

> **4주간의 루틴 완성으로 평생 자기주도 학습 습관이 형성됩니다**
> 매주 **월수금 3회** 방문으로 **월 12회** 완주 시, 아이의 영어 실력과 자신감이 달라집니다!

## 5가지 챌린지 유형

| 챌린지 | 내용 | 교재 |
|--------|------|------|
| **원서 리딩 챌린지** | 시즌별 원서 + 활동지 | 자체 워크시트 |
| **뉴스 챌린지** | 영어 기사 읽기 + 요약 | 자체 워크시트 |

> **핵심 포인트**: 매달 새로운 테마와 함께 선택한 챌린지 **자체 워크시트**로 진행

## 챌린지별 자기진단표

> "우리 아이, 어떤 루틴이 필요할까요?"
> 아래 5가지 영역에서 체크된 항목이 **2개 이상**인 영역이 우리 아이에게 **가장 필요한 루틴**입니다!

### 원서 리딩 챌린지가 필요한 아이
- 책을 읽어도 줄거리 이해가 잘 안 된다고 해요
- 한 권을 끝까지 읽는 데 시간이 오래 걸려요
- 읽은 책 내용에 대해 이야기하거나 정리하는 게 어려워요
- 질문을 받으면 "그냥 재미있었어요"로만 대답해요

→ **원서 리딩 루틴으로 매일 짧게 읽고 말하기 + 쓰기 훈련**

### 단어 챌린지가 필요한 아이
- 단어 시험은 잘 보는데, 금방 잊어버려요
- 외운 단어를 문장에서 활용하지 못해요
- 같은 단어를 여러 번 다시 틀려요
- 받아쓰기를 힘들어하거나, 철자 실수가 잦아요

→ **단어 루틴으로 쓰기 + 문장 연결 + 반복 확인**

### 뉴스 챌린지가 필요한 아이
- 짧은 글도 핵심을 파악하는 데 시간이 걸려요
- 한 문장 안에서 중요한 내용을 놓치는 경우가 많아요
- 아이가 세상 돌아가는 일이나 뉴스에 흥미가 없어요
- 요약하거나 의견 쓰기를 어려워해요

→ **뉴스 루틴으로 키워드 찾기, 요약훈련, 배경지식 확장**

### 문장쓰기 챌린지가 필요한 아이
- 한 문장을 스스로 만들기 어려워해요
- 단어는 알지만 문장으로 연결을 못 해요
- 쓰는 걸 두려워하거나 막막해해요
- 글쓰기 시간이 너무 짧거나, 딱 한 문장만 씁니다

→ **쓰기 루틴으로 매일 짧은 문장부터 확장 훈련**

### 문법 챌린지가 필요한 아이
- be동사, 일반동사를 자꾸 헷갈려요
- 문장을 고쳐 써도 같은 실수를 반복해요
- 문법 용어는 아는데 실제 문장에서 잘 못 써요
- 자꾸 같은 실수를 해서 쓰기를 싫어해요

→ **문법 루틴으로 정확하게 쓰는 연습**

## 영어 노출과 성장 그래프

> **30~40분 × 12회 루틴!**

- 한 달 **12회(주 3회, 30~40분)**만으로 **6~8시간**의 집중 영어 노출
- 짧고 꾸준한 루틴이 **기억력·문장력·자신감**을 비약적으로 끌어올림
- 짧고 꾸준한 학습(매일 30분)이 장기성과에 더 효과적이라는 **연구 결과** 다수

### 연구 데이터
- 하루 30분씩 꾸준히 공부한 학습자들은 3개월 만에 말하기와 이해력이 **50% 이상** 향상
- 끊김없는 학습 루틴을 유지한 학생들의 **90%**가 영어 성적 상위권 유지
- (출처: 교육부 학습효과 연구, 2023)

### 성장 곡선 (영어 실력 향상도 / 자신감 지수)

| 시점 | 향상도 |
|------|--------|
| 1주차 | 약 10% |
| 2주차 | 약 15% |
| 3주차 | 약 25% |
| 4주차 | 약 30% |
| 8주차 | 약 55% |
| 12주차 | 약 75% |

> **"쉽게 깨지지 않는 습관이 성적과 실력을 만든다!"**
