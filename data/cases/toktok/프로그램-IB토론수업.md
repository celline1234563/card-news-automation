# 톡톡 잉글리쉬 — IB 기반 토론형 수업 프로그램
> 핵심 수업 구조 및 단계별 문제 진단 정리

---

## 수업 컨셉
- **온라인 학교 IB 기반** → 읽어온 베이스로 토론형 수업 진행
- 유연한 사고, 에세이, 영어 논술을 다루는 학원
- 반 배정 기준: **학년 + SR 지수 + 레벨 + 성향 + 학부모가 부족하다고 느끼는 부분**

---

## 톡톡이 진단하는 단계별 문제

> "영어를 언어적으로만 잘한다고 영어를 잘하는 걸까? 영어도 기본적인 '언어'의 영역. 결국 논리력이 핵심."

### 1단계 — 유치원 수준

| 영역 | 문제점 | 톡톡 해결법 |
|------|--------|-----------|
| **영어** | 어휘 부족, 단어를 기계식으로 외워서 설명 불가. 리딩 이해도·발화·영작 현저히 부족. "많이 읽으면 된다"는 접근으로 밸런스 깨진 상태 | 단어를 입으로 설명하는 수업, 동의어·반의어 파악, 리딩 후 자신의 생각을 세 줄 이상 설명 |
| **논리** | "좋아요! 싫어요!" 수준. 상태 파악, 사실과 의견 분리 불가. Fact vs Opinion 구분 힘듦 | 사실과 의견 분리 훈련 |

### 2단계 — 초등학교 수준

| 영역 | 문제점 |
|------|--------|
| **영어** | 1단계와 비슷하지만 말하면 알아듣는 수준. 단어 설명 가능, 문장 이해는 함 |
| **논리** | 최소 2문장 이상 설명 가능. 하지만 근거·논리 부족, 문맥이 안 맞고 대체로 근거가 없음 |

### 3단계 — 중학교 수준

| 영역 | 문제점 |
|------|--------|
| **영어** | 의사소통에 문제 없지만, 사고를 이야기할 때 딜레이 발생. 영어를 잘하긴 하지만 깊이 부족 |
| **논리** | 에세이가 나오긴 하지만 **나의 생각이 없는 상태**. 요약만 있는 에세이. 질문하면 답하지만 직접 쓸 때 사고가 튀어나오지 않음. 나열식이 많고 팩트 인용은 가능하지만 **나만의 사고는 불가능**. 설득력 부족 |

### 4단계 — 고등학교 수준

| 영역 | 문제점 |
|------|--------|
| **영어** | 원어민급 수준이지만 **깊이가 얕다**. 영어로 살아가는 데 문제는 없음 |
| **논리** | 독서량 현저히 부족. 시사·경제·정치 지식 없음. 본인의 사고는 이야기할 수 있지만 철학적이거나 깊이가 없음. **유아적 근거에 머물러 있다**. 고급 어휘 사용한 통찰력 부족, 다면적 사고 불가능 |

## 해결 포인트 — 톡톡의 핵심

> **"수정까지 해야 된다!"** — 수정을 끝까지 할 수 있도록, 결국 내 실력이 되도록 하는 힘

- "해보면 어떨까요?" 식의 **제안형 피드백**
- 학생에게 맞는 과목이나 **사고 확장을 위한 내용을 계속 안내**

## 수업 구조

### 기본 수업
- IB 기반 → 읽어온 자료로 **토론형 수업** 진행
- 유연한 사고 + 에세이 + 영어 논술

### 관리 서비스 — 주 1회 수업이지만 관리는 매일

#### 피드백 방식
- **문법 수정** + **사고력 질문** ("이런 것도 생각해 볼 수 있어?" 식 질문이 많음)
- 수업 후 여러 방면으로 리서치·고민하도록 유도 → 지문에 녹아드는 것이 달라짐

#### 피드백 단계
| 단계 | 내용 | 효과 |
|------|------|------|
| 요약 피드백 | 문법 위주 수정 | 기본 영작 능력 향상 |
| 사고력 피드백 | 사고력 질문에 대한 답변 유도 | 에세이가 탄탄해지고 실력 향상 |

#### 주간 학습량
| 항목 | 횟수 |
|------|------|
| 수업 | 8번/월 |
| 숙제 제출 | 12번/월 |
| 리뷰 테스트 | 4번/월 (매주) |

#### 숙제 프로세스
1. 워크시트 작성
2. 사진 촬영 + 제출
3. **음성 녹음** — 단어 설명 및 내용 발화
4. **수정 완료해야 숙제 인정** (수정 없이는 미완료)

> "수업은 끝났는데 또 와요. 근데 하니까 늘어요."

#### 자동 관리 시스템
- 월별 평가 자동 생성
- 녹화된 수업 자동 제공
- 테스트·숙제 제출 체크리스트

## 카드뉴스 활용 포인트

### 숫자 소재
- "주 1회 수업, 월 **12번** 숙제, 매주 리뷰 테스트"
- "워크시트 → 제출 → 음성녹음 → **수정까지** 완료해야 숙제 인정"
- "4단계 진단 — 영어 + 논리를 동시에 잡는 커리큘럼"

### 스토리 소재
- "에세이에 '나의 생각'이 없습니다 — 요약만 있는 글을 쓰고 있지 않나요?"
- "원어민급인데 깊이가 얕다 — 톡톡이 진단하는 4단계 문제"
- "수정을 끝까지 해야 내 실력이 됩니다"
- "문법 수정 + 사고력 질문 = 톡톡 피드백의 비밀"
- "좋아요! 싫어요!에서 → 다면적 사고까지"

### 차별화 포인트
- **IB 기반 토론형 수업** — 단순 영어가 아닌 논리·사고력 동시 향상
- **4단계 문제 진단** — 영어 실력 + 논리력을 분리해서 진단
- **수정 완료형 숙제** — 제출만으로 끝나지 않는 관리 시스템
- **음성 녹음 숙제** — 단어를 설명하고 발화하는 훈련
- **사고력 피드백** — 문법만 고치는 게 아니라 "이것도 생각해 봐" 질문
